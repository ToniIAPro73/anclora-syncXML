# Roadmap

## Current phase: pre-MVP / controlled validation

Focus:

- Privacy-first import and validation.
- XML generation.
- Safer local-first workflows.
- Controlled pilot feedback.

## Near-term priorities

- Improve README and contributor onboarding.
- Add CI checks for lint, typecheck, tests and build.
- Expand XML validation tests.
- Improve synthetic fixtures.
- Review legal/disclaimer wording.
- Improve accessibility.

## Mid-term priorities

- Better mapping workflow for external spreadsheet formats.
- Stronger audit trail without PII.
- Optional encrypted persistence.
- More complete i18n coverage.
- Better developer documentation.

## Out of scope for now

- Legal advice.
- Production compliance guarantee.
- Automatic official submission without audit.
- Processing real guest data in public examples.
