## Nota operativa

El repositorio actual de SyncXML contiene sobre todo validacion determinista, UX defensiva y automatizacion controlada. Este registro separa:

- features no IA ya implementadas;
- capacidades IA previstas;
- integraciones auxiliares fuera del core del repo.
