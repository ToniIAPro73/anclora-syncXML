# Criterios de Éxito del Piloto — Anclora SyncXML

Para considerar que el piloto con un participante ha sido exitoso, se deben cumplir los siguientes hitos:

## 1. Éxito Operativo
*   **Importación Exitosa**: El usuario ha podido subir su propio Excel/CSV (ajustado a la plantilla o manual) sin errores de sistema.
*   **Detección de Valor**: El usuario ha identificado al menos un error real en sus datos sintéticos gracias a las reglas de validación de SyncXML.
*   **Paquete Local**: El usuario ha descargado y comprendido el contenido del paquete de conservación.

## 2. Éxito de Privacidad
*   **Confianza Confirmada**: El usuario expresa que el procesamiento temporal y el no almacenamiento permanente le aportan tranquilidad legal/operativa.
*   **Enmascaramiento Útil**: El usuario confirma que la vista previa enmascarada es suficiente para su revisión diaria.

## 3. Éxito Comercial
*   **Feedback de Pago**: El usuario indica una disposición de pago (SaaS o por volumen) superior al coste de operación del servicio low-cost.
*   **Petición de Conectores**: El usuario identifica qué conectores externos (PMS, Channel Manager) le harían pasar de un piloto manual a una suscripción recurrente.
