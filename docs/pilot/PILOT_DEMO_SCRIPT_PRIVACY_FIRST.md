# Guion de Demostración del Piloto — Anclora SyncXML

## Escenario: Revisión Privacy-First de Reserva con Excel

### 1. Introducción y Posicionamiento
*   Muestra la **Landing Page**.
*   Explica que SyncXML es una *capa ligera de revisión*, no un PMS.
*   Enfatiza la **Privacidad por Defecto**: *"Tus datos se procesan en memoria y no se guardan permanentemente"*.

### 2. Acceso y Seguridad
*   Haz clic en *"Iniciar sesión"*.
*   Muestra el aviso de *"Piloto Controlado"* y la necesidad de credenciales autorizadas.

### 3. Carga y Revisión
*   Importa un archivo **Excel o CSV sintético**.
*   Muestra la **Vista Previa Enmascarada**: *"Como ves, los documentos y emails están protegidos incluso para el operador"*.
*   Explica la **Detección de Errores**: Corrige un código de municipio o una fecha inconsistente directamente en el flujo.

### 4. Generación y Conservación
*   Genera el **XML Revisable**.
*   Muestra los **Estados Prudentes**: *"El sistema indica que el XML está generado y listo para tu revisión, marcado como Solo Piloto"*.
*   Descarga el **Paquete Local de Conservación**: Abre el ZIP y muestra el `README_conservacion.txt` con los avisos legales.

### 5. Cierre y Borrado
*   Pulsa en **"Borrar datos de esta operación"**.
*   Muestra que la sesión queda limpia: *"Privacidad cumplida: no queda rastro de la operación en nuestros servidores"*.
