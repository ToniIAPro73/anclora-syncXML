# Knowledge Vault — Anclora SyncXML

Este directorio almacena material local de gestión del conocimiento relacionado con Anclora SyncXML.

Incluye:
- snapshots del repo en formato .txt;
- documentos derivados de NotebookLM;
- auditorías;
- diagnósticos;
- matrices de prioridades;
- planes de acción;
- registros de riesgos;
- índices maestros;
- changelogs ejecutivos;
- documentos preparados para una futura Gema/GPT.

Este directorio no contiene código de producción. Su objetivo es preservar contexto, versiones, análisis y decisiones.

## Reglas de uso

1. El repo real sigue siendo la fuente técnica principal.
2. Los snapshots .txt son referencias históricas.
3. Los documentos refinados sustituyen a los diagnósticos iniciales.
4. No se deben mezclar informes de mercado con documentos técnicos salvo en carpetas puente.
5. Cada actualización importante del producto debe generar un changelog ejecutivo.
6. Antes de subir documentos a NotebookLM o a una Gema/GPT, deben estar refinados y auditados.
